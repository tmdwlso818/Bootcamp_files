9. 확인 불가 / 한계 항목

  • 시연 영상의 실제 동작 내용은 정의서 5.4에 따라 Agent가 판정하지 않았다. 요구사항서의 타임스탬프 제공 여부만 표로 정리했다.
  • R-33의 "이미 실행 중인 로컬 LLM 서버 재사용"은 코드 경로는 확인했지만, 별도 실제 LLM 서버를 먼저 띄운 재사용 시나리오는 수행하지 않았다.
  • R-61~R-63 debug/release workflow는 코드 경로를 확인했지만, 전체 리포트 생성이 LLM 호출을 포함하므로 별도 장시간 실행 검증은 하지 않았다.
  • 샌드박스 내부에서는 로컬 포트 바인딩이 PermissionError: [Errno 1] Operation not permitted로 차단되어, 앱 기동 HTTP 검증은 승인된 권한 상승 실행으로 수행했다.
  • ./build.sh 중 npm/Hugging Face DNS 실패 경고가 있었지만 CMake가 stale UI assets를 사용해 최종 빌드와 테스트는 성공했다. 네트워크가 완전히 필요한 신규 환경에서는 이 경고가 재현성 리스크가 될 수 있다.
